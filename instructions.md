# Goal
SPA application for generating a character build for the CYOA 'Accidental Magical Girl'.

# Requirements
- User-friendly interface
- Ability to customize character attributes
- Integration with CYOA game mechanics
- Responsive design for various screen sizes

# Technologies
- React.js
- Redux
- Jest

# Development Plan
- Define project structure and architecture
- Implement user interface components
- Create Redux store and actions
- Write unit tests using Jest
- Implement CYOA game mechanics
- Integrate with data.js and coin-spend.txt
- Test and debug application
- Deploy to production

# CYOA Info
- build is generated by rolling 11d20 and assigning the result of each roll to the character's choices
- data for choices is in data.js
- coin spend logic is in coin-spend.txt
